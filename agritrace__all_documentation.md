# AgriTrace+

## Project Description
I want to build a web-based AgriTrace+: A Digital Livestock Registration and Public Reporting with Geo-Tagging Integration. I want this web has a home page that has a header home, about, contact, and login button that has content inside of each, then after when it click the login button it will show the login popup that has content of input email and password, login button, register account popup, forgot password popup, and public report button popup. then after register and login i want a three panels which is Admin panel for controlling all the content and functions of other panel has a live map that shows the farm location and density of animals based on the map legend 
Green: Low Density (10-20),Yellow: Medium (20-30),Red: High Density (>30),Blue: Incident Alert has user management can create users has data management for database can import export of data has  activity log when they import and export data has backup database and restore from backup when it lost the data and i want this panel has a reports & analytics of all livestock farm and reporting incident, DA Officer panel has a control of the accepting the registrations of the farmer has a live map that shows the farm location and density of animals based on the map legend Green: Low Density (10-20),Yellow: Medium (20-30),Red: High Density (>30),Blue: Incident Alert has farm inspection if the farm registration are legit owned of real farmer and incident management there is a disease outbreaks and public report if someone report an incident that can cause affect in biosecurity has a reports & analytics of livestock farm and public report can export the files has a profile button information about the official if there are legit and can edit details like the profile pictures make it popup when clicking edit button has a functions to change password, and farmers panel has a function of registering livestock farm quantity or density of animal locations of farm and can report any kind of problems like disease  has a profile button information about the official if there are legit and can edit details like the profile pictures make it popup when clicking edit button has a functions to change password has livestock health monitoring live using any kinds of arduino sensor has notifications of alerts and reports from admin and DA.
